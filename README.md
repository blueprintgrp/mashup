## Contributing

#### Cloning

You can clone the mashup repository like underneath. To contribute you could create a pull request or fork the repository.

```bash
git clone https://github.com/blueprintgrp/mashup.git 
cd mashup
npm install --save-dev
```

#### Running

```bash
npm run dev
```

#### Building

```bash
npm run build
```
